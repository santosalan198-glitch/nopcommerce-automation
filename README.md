# NopCommerce Automation Framework

Selenium + TestNG automation framework for SauceDemo

## Tech Stack
- Java 11
- Selenium 4.21
- TestNG 7.10
- Maven
- Allure Reports
- GitHub Actions CI/CD

[![Allure Report](https://img.shields.io/badge/Allure-Report-brightgreen)](https://santosalan198-glitch.github.io/nopcommerce-automation/)

- LoginTest
- InventoryTest
- CartTest

## Run locally
```bash
mvn clean test
mvn allure:serve
```
